Ⅳ. Graph Structures

Ⅰ. 인접 리스트 그래프 (Adjacency List Graph)


정의: 객체 간의 복잡한 관계를 정점(Vertex)과 이들을 잇는 간선(Edge)으로 표현하는 비선형 자료구조입니다.
장단점: 세상의 모든 복잡한 관계(인맥, 지도 등)를 있는 그대로 표현할 수 있을 만큼 유연하지만, 트리나 리스트에 비해 알고리즘의 시간 복잡도가 높고 구현이 까다롭습니다.
사용 사례: 내비게이션의 최단 경로 탐색, 소셜 네트워크 서비스의 친구 추천 시스템, 검색 엔진의 웹 페이지 랭킹 알고리즘 등에 필수적으로 활용됩니다.
